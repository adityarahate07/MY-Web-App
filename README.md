# 🌐 Professional Multi-Feature Web App

## 📌 Project Description

This is a **Professional Web Application** built using **HTML, CSS, and JavaScript**.
It combines multiple interactive features into a single responsive website.

---

## 🚀 Features

* 🎯 Change Text Button
* 🌙 Dark Mode Toggle
* 🔢 Counter App (+ / -)
* 🎨 Random Background Color
* 👁️ Show / Hide Text
* 🙋 Greeting App (User Input)
* ➕ Simple Calculator

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
